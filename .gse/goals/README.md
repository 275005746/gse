# Goal Details

Use this folder for module-level or stream-level goal details when `.gse/goal-map.md` becomes too large.

Keep `.gse/goal-map.md` as the short index:

- North Star
- Current focus
- Top goal nodes
- Risks
- Next slices

Put detailed module goals here, for example:

- `frontend.md`
- `backend.md`
- `memory.md`
- `worker.md`
- `release.md`
