# Tooling

Minimum tools: git, shell, Node.js, and project test/build commands.

Recommended optional tools:

- `rg` / `rg --files` for fast search.
- LSP or code index for large projects.
- Browser or Playwright for UI verification.
- Host-specific agents or subagents when the host proves they are available.
- Project learning files for durable lessons.

Do not claim a tool was used when it was not available.
